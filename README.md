# 中关村学院 · AI 博士生模拟器

> 欢迎来到「北京中关村学院」。学院践行 **「极基础、极应用、极交叉」** 的科研理念，以 **项目制、超常规培养** 为主线——你的博士长征，从这一份 README 外的浏览器标签页启程。

这是一款 **浏览器里的文字模拟经营小品**：你扮演一名主攻 **人工智能与交叉学科** 的直博生，在 **16 个季度**（约四年学制）里选课、跑实验、写论文、出差、混课题组、和导师与同门周旋，并尽量在 **理智、健康、精力、声望与不端嫌疑** 之间活到答辩那一天。没有 3D 大作，只有进度条、弹窗和日志里那些 **「Loss 曲线忽上忽下」「培养办已发出书面警告」** 式的真实感。

## 你在学院里要做什么

- **档案**：看状态条，摸鱼、散步、撤稿止损；**资金与算力**也在这里盯着，别等透支了才想起看账户。
- **主页**：日常里修课、讲坛、助教；科研里文献、实验、撰稿、出差报告——每季 **行动次数有限**，用完就得点 **「进入下个季度」** 去领补助、挨审稿、碰随机事件。
- **团队**：拜师前可以 **拜访求教** 刷意向好感；入组后要互动、团建，不然好感下滑，课题组气氛会反过来 **拖慢你的科研条**。
- **动态**：学术日志记仇，朋友圈点赞回血；前两年每学期记得把 **「校园与讲坛」** 凑够次数，培养办真的会记台账。

部分叙事与随机文案可走 大模型 API（可选）；不配 key 也能玩，只是少了一点由模型现编的骚话。

## 给技术官的附记（本地运行）

```bash
npm ci
npm run dev
```

可选：复制 `.env.example` 为 `.env` 并填写 `API_KEY`（构建时注入为 `process.env.API_KEY`）。GitHub Pages 请在仓库 **Settings → Secrets** 中新增同名 Secret。

## 在线体验（GitHub Pages）

推送 `main` 或 `master` 后，由 [GitHub Actions](.github/workflows/deploy-github-pages.yml) 自动构建部署。

1. 仓库 **Settings → Pages**：Source 选 **GitHub Actions**。
2. 部署完成后访问：`https://<你的用户名>.github.io/<仓库名>/`

仓库改名不必改代码，工作流会按新仓库名设置资源路径。
